Bem vindo ao meu site

Esta é a página inicial, se trata de uma página acadêmica, para a disciplina de Desenvolvimento Web.

Desenvolvimento web é o processo de criar e manter websites, aplicações web e outros conteúdos para a internet, ele engloba tanto a parte visual (front-end) como a funcionalidade e lógica (back-end), os profissionais nessa área são chamados de desenvolvedores web e podem especializar-se tanto e áreas como front-end, back-end ou até mesmo full-stack.
